Method

Interactive plots for EP300-ZNF384 gene expression in pre-B acute lymphoblastic leukaemia (pre-B ALL).

These plots were generated as Supplementary Files using R (version 3.4.1) (2017-06-30) using Glimma_1.4.0 package.
